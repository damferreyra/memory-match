    Rewrite GameScene.ts (currently a placeholder) to implement a full grid rendering. The class must have typed properties for phase 4 readiness.

    Class structure:
    ```typescript
    import Phaser from 'phaser';
    import { GAME_WIDTH, GAME_HEIGHT, SCENE_KEYS } from '../constants';
    import { CARD_WIDTH, CARD_HEIGHT, CARD_CORNER_RADIUS, CARD_BACK_COLOR, CARD_BACK_BORDER, CARD_FACE_BG, CARD_FACE_BORDER, SYMBOLS, SYMBOL_COLORS, SYMBOL_FONT_SIZE } from '../config/cards';
    import { GRID_LAYOUT, getCardPosition } from '../config/grid';
    import { CARD_DEPTH } from '../config/ui';
    import { CardData, generateCardPairs } from '../game-logic';

    export class GameScene extends Phaser.Scene {
      private cards: CardData[] = [];
      private containers: Phaser.GameObjects.Container[] = [];

      constructor() { super({ key: SCENE_KEYS.GAME }); }

      create(): void {
        this.cards = [];
        this.containers = [];
        this.drawBackground();
        const symbolIds = generateCardPairs(8);
        this.buildGrid(symbolIds);
      }

      private drawBackground(): void { ... }
      private buildGrid(symbolIds: number[]): void { ... }
    }
    ```

    drawBackground(): Copy the exact gradient call from MenuScene: `bg.fillGradientStyle(0x1a1a2e, 0x1a1a2e, 0x16213e, 0x16213e, 1)` then `bg.fillRect(0, 0, GAME_WIDTH, GAME_HEIGHT)`.

    buildGrid(symbolIds): Loop i from 0 to 15. For each card:
    1. Push a CardData to this.cards: `{ id: i, symbolId: symbolIds[i], state: 'faceDown' }`
    2. Get position: `const pos = getCardPosition(i, GRID_LAYOUT)`
    3. Create container: `const container = this.add.container(pos.x, pos.y)` then `container.setDepth(CARD_DEPTH)`
    4. Draw back Graphics: fillStyle(CARD_BACK_COLOR), fillRoundedRect(-CARD_WIDTH/2, -CARD_HEIGHT/2, CARD_WIDTH, CARD_HEIGHT, CARD_CORNER_RADIUS), lineStyle(2, CARD_BACK_BORDER, 1), strokeRoundedRect same args.
    5. Add '?' text: `this.add.text(0, 0, '?', { fontSize: '48px', fontFamily: 'Arial, sans-serif', color: '#ffffff' }).setOrigin(0.5)`
    6. Draw front Graphics (hidden): fillStyle(CARD_FACE_BG), fillRoundedRect same args, lineStyle(2, CARD_FACE_BORDER, 1), strokeRoundedRect same args, then `.setVisible(false)`
    7. Add symbol text (hidden): use `SYMBOLS[symbolIds[i]]` for the character. Color: `'#' + SYMBOL_COLORS[SYMBOLS[symbolIds[i]] as Symbol].toString(16).padStart(6, '0')`. fontSize: `${SYMBOL_FONT_SIZE}px`. setOrigin(0.5). setVisible(false).
    8. container.add([back, questionMark, front, symbolText])
    9. Push container to this.containers

    CRITICAL rules:
    - Container children use coordinates RELATIVE to the container origin. The container is already at pos.x, pos.y. Draw all rects at (-CARD_WIDTH/2, -CARD_HEIGHT/2), text at (0, 0).
    - fillStyle() and lineStyle() require hex integers — use the constants directly, never CSS strings.
    - SYMBOL_COLORS keys are `Symbol` type (the project type alias) — cast with `as Symbol` when indexing.
    - No literal numbers in scene code — every number must come from a named constant.
    - Symbol text color conversion at call site: `'#' + SYMBOL_COLORS[sym as Symbol].toString(16).padStart(6, '0')`.

    Run `npm run lint` after writing. Fix any Biome errors (tabs, single quotes).
